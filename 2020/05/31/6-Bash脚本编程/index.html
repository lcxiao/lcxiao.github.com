脚本的状态返回值：

默认是脚本执行的最后一条命令的返回值；
自定义状态退出状态码:
exit [n]：n为自己指定的状态码；
注意：SHELL进程遇到exit时，即会终止，因此整个脚本执行即为结束。

过程式编程语言的代码执行顺序：
顺序执行： 逐条执行；
选择执行：
代码有一个分支： 条件满足时才会执行；
两个或以上的分支：只会执行其中一个满足条件的分支；
循环执行：
某代码片段（循环体）要执行0、1或者多个来回；
选择执行：
单分支的if语句：
if 测试条件
then
代码分支
fi

双分支的if语句
if 测试条件；then
条件为真时执行的分支
else
条件为假时执行的分支
fi
